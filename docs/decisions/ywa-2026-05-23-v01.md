# YWA 2026-05-23-v01 Decisions

Decision log for the Yuba Water Agency submission standardized by
`scripts/clean-ywa-2026-05-23-v01.R`.

## Decisions

- Standardized output targets `RestorationProjectSubmission`; submitted
  `project_id` and `update_date` are omitted as program-assigned
  canonical-record fields.
- The submitted GeoPackage layer is used for geometry and attributes. The
  companion workbook project tabs are inventoried as supplemental submitted
  attributes but are not joined into the standardized output.
- Source field `constructon_completion_year` is treated as the misspelled source
  name for `construction_completion_year`.
- Submitted semicolon-delimited `contact_name` and `contact_email` values are
  reduced to the first listed contact/email to satisfy the schema's singular
  primary-contact field.
- Submitted `early_implementation` value `True` is standardized as `TRUE`.
- Submitted `target_species` value `Steelhead Trout` is standardized as
  `Steelhead trout`.
- Missing submitted acreage values are retained as missing rather than replaced
  with geometry-derived acreage.
