# DWR 2026-06-19-v01 Decisions

Decision log for the DWR submission standardized by
`scripts/clean-dwr-2026-06-19-v01.R`.

## Decisions

- Standardized output targets `RestorationProjectSubmission`; `project_id` and
  `update_date` are omitted as program-assigned canonical-record fields.
- Source did not include `funding_gap`; where both `estimated_budget` and
  `funding_secured` were submitted, `funding_gap` is derived as
  `estimated_budget - funding_secured`.
- Submitted semicolon-delimited values are preserved but normalized to use a
  single space after each semicolon, and empty submitted list elements are
  dropped.
- Submitted acreage values are preserved rather than replaced with
  geometry-derived acreage.
- Missing required values are left missing and flagged in validation rather
  than inferred from context.
