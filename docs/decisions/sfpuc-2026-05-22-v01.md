# SFPUC 2026-05-22-v01 Decisions

Decision log for the San Francisco Public Utilities Commission submission
standardized by `scripts/clean-sfpuc-2026-05-22-v01.R`.

## Decisions

- Standardized output targets `RestorationProjectSubmission`; submitted
  `project_id` and `update_date` are omitted as program-assigned
  canonical-record fields.
- The submitted GeoPackage contains two layers with matching feature count and
  fields. The shorter `merged` layer is used as the source layer.
- The source attribute field `geometry` contains missing values and is omitted.
  The active spatial geometry is preserved from the GeoPackage geometry column.
- Submitted `project_name`
  `Upstream of Old LaGrange Bridge Restoration Projec` is treated as apparent
  field-width truncation and standardized as
  `Upstream of Old LaGrange Bridge Restoration Project`.
- Submitted lead-entity text `San Francisco Public Utilites Commission` is
  treated as an apparent spelling error and standardized as
  `San Francisco Public Utilities Commission`.
- Submitted funding-source text is converted from comma-delimited acronyms to
  semicolon-delimited values. `SPFUC` is treated as an apparent typo for
  `SFPUC`.
- Submitted semicolon-delimited values are preserved but normalized to use a
  single space after each semicolon.
- Submitted acreage values are preserved rather than replaced with
  geometry-derived acreage.
- Missing `acreage_fish_food` values are retained as missing for now.
