# SCWA 2026-05-22-v01 Decisions

Decision log for the Solano County Water Agency submission standardized by
`scripts/clean-scwa-2026-05-22-v01.R`.

## Decisions

- Standardized output targets `RestorationProjectSubmission`; workbook
  `project_id` and `update_date` are omitted as program-assigned
  canonical-record fields.
- The submitted zip is extracted to a temporary directory at runtime. Raw files
  under `data-raw/` are not modified.
- Workbook rows are treated as the attribute source and joined to named
  shapefiles by project name.
- The 91 submitted gravel point features are aggregated into one multipoint
  geometry for `Putah Gravel Augmentation and Scarification`; the submitted
  point locations are preserved as point components.
- Individual gravel point attributes are intentionally ignored. The workbook
  provides the project/group observation attributes used for the standardized
  gravel project record.
- The submitted Nishikawa polygon is repaired with `sf::st_make_valid()` because
  it is invalid as submitted.
- Workbook literal `null` values are treated as missing values.
- Missing construction years for `Putah Creek fish passage at Los Rios Check Dam`
  and `Putah Creek fish passage at County Rd 106a` are retained as missing
  values for now.
- Missing `funding_gap` values are derived as
  `estimated_budget - funding_secured` when both inputs are present.
- Submitted acreage values are preserved rather than replaced with
  geometry-derived acreage.
